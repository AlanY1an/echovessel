# Code of Conduct

EchoVessel adopts the **Contributor Covenant, version 2.1** as its code of
conduct, without modification. The full text — covering the pledge, expected
behaviour, scope, enforcement guidelines, and consequence ladder — lives at the
canonical URL:

> <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

The translations linked from that page are also considered authoritative for
this project.

maintainers are obligated to respect the privacy and security of the reporter
of any incident, per the Contributor Covenant's enforcement guidelines.

## Scope

This code applies to every project space — the GitHub repository (issues, pull
requests, discussions), any official chat or community channel that may exist
in the future, and to anyone representing the project in public spaces.
